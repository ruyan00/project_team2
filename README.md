# 2조 프로젝트(4반) - To-Do 리스트

[프로젝트 소개]

C언어를 활용하여 제작한 To-Do 리스트 프로그램입니다. 사용자가 할 일을 등록, 조회, 완료 및 삭제할 수 있습니다.

[주요 기능]
1. 할 일 조회
2. 할 일 추가
3. 할 일 완료
4. 할 일 삭제
